# conv-brute
wifi bruteforce Script with wordlist and without wordlist ( made for kali linux)

### Usage:
```
git clone https://github.com/FDX100/conv-brute.git
cd conv-brute
chmod +x conv-brute.py
python conv-brute.py
```
